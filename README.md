# react-redux-boilerplate

A boilerplate of all things in one file.

详见：[React + Redux 最佳实践](https://github.com/sorrycc/blog/issues/1)

> Quickly prototype with one file, then slowly grow the app.

## Setup

```bash
$ npm install
```

## Running

```bash
$ npm start
```

## Build

```bash
$ npm run build
```
