# react-redux-boilerplate

A boilerplate with react, redux, redux-saga ...

详见：[React + Redux 最佳实践](https://github.com/sorrycc/blog/issues/1)

## Setup

```bash
$ npm install
```

## Running

```bash
$ npm start
$ open http://localhost:8989/
```

## Build

```bash
$ npm run build

// Run build without compress
$ npm run build -- --no-compress
```
